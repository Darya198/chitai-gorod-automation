## Автоматизация тестирования интернет-магазина «Читай-город»

Данный проект является дипломной работой по курсу «Инженер по тестированию». В рамках проекта автоматизированы ключевые UI и API сценарии сайта "Читай-город" (https://chitai-gorod.ru).

## Используемые технологии
* Язык: Python 3.x
* Фреймворк: Pytest
* UI: Selenium + Webdriver Manager
* API: Requests
* Отчетность: Allure
* Линтинг: Flake8 (соответствие PEP 8)
* Паттерн: Page Object Model (POM)

## Инструкции по запуску тестов

### 1. Подготовка окружения
Установите все необходимые зависимости:
pip install -r requirements.txt

### 2. Запуск тестов через маркеры
Запуск всех тестов (UI + API): pytest
Запуск только UI-тестов: pytest -m ui
Запуск только API-тестов: pytest -m api

Для просмотра логов (print) добавьте флаг -s

### 3. Генерация отчета Allure
Сбор данных: pytest --alluredir=allure-results
Генерация и открытие отчета в браузере: allure serve allure-results

## Конфигурация проекта
Все специфичные данные (URL, токены, тестовые ID) вынесены в файл config.py для удобства настройки окружения без изменения кода тестов.
